# Git Ruler

Use concise Conventional-style commit messages:

```bash
feature(scope): summary
fix(scope): summary
docs(scope): summary
chore(scope): summary
```

Examples:

```bash
feature(repo-argument): adding repo argument
docs(task-flow): formalize pidocker workflow
```
